Dada SDK for Golang

## 安装
##### 启用 Go module

```
go get github.com/royromny/dada
```

```
import github.com/royromny/dada
```

##### 未启用 Go module

```
go get github.com/royromny/dada
```

```
import github.com/royromny/dada
```
